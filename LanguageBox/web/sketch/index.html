Как работает сервис

Репетитор или методолог школы английского языка заносит в сервис контент который помогает лучше изучать английский язык для ученика. Ученик использует сервис для изучения. Весь контент ему доступен онлайн и на мобильных устройствах. Контент может быть: аудио, видео, тесты и т.д.

Для кого создан сервис

Для учеников

Ученики могут изучать английский язык онлайн через компьютер, смортфон, планшет. Они могут сохранять пройденный материал в своем личном кабинете. Проходить тесты. Измерять рост знаний.

Для репетиторов

Вся библиотека контента в одном месте. Можно составлять свою методику обучения. Можно раздавать дополнительные задания ученикам в два клика. Для ученика становиться удобнее заниматься с репетитором так как есть доступ к заданиям онлайн.

Для языковых школ

Методолог языковой школы дополняет методику онлайн контентом. После этого ученики языковой школы могут изучать английский язык онлайн через: компьютер, смортфон, планшет. Они могут сохранять пройденный материал в своем личном кабинете. Проходить тесты. Вся библиотека дополнительных заданий в одном месте. Можно улучшать стандартные коммуникативные методики обучения. Среда позволяет геймифицировать процесс обучения посредством рейтингов с другими учениками группы.
